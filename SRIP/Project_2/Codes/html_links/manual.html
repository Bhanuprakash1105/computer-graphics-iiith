MANUAL
  • After starting the experiment we need to enter the frame-width and frame-height as input and click on the 'Enter' button.

  • The min and max values of frame-width and frame-height are (min: 0 and max: 25).

  • Or else click on the 'Start With Default Values' button by which the simulation will run on default values.

  • If the inputs are given by the user then enter the 1st end point and 2nd end points of the line and click on the 'Sart' button for proceeding further or else click on the 'Back' button for going back.

  • After clicking 'Start' the user can run the Rasterization line simulation by clicking on the 'Next Iteration' and 'Previous Iteration' buttons or else click on 'Back' to go back.

  • If the default simulation is choosen then use 'Next Iteration' and 'Previous Iteration' buttons for observing the output.

  • In the default simulation frame-width = 15 and frame-height = 15.

  • In the default simulation the 1st end point is (2, 5) and 2nd end point is (9, 10).
