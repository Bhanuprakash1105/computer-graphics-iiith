MANUAL

After clicking the Start we can observe the content of World Tab in which we can observe the description of the experiment.

Below we can observe World and Display buttons through which we can shift between the world and display tabs.

In the diplay tab many features are available:

  1. We have Draw grid XY, YZ and ZX to choose as check boxes and we can choose as per our choice.

  2. The Draw grid XY check box has default checked value but user can choose to check it or not.

  3. The distance between two intersection points on the line is 1.0 units in all the planes.

  4. We have Show Animation and Remove Animation buttons by which we can see the animation provided and undo the animation.

  5. Transistion feature are available by which we can shift the 3D model to various coordinates.
    ** Please start the transition with (x, y, z) = (0.5, 0, 0) units for avoiding refershing of the page **.

  6. Remember that the x, y and z values which you enter must be the units which will be distance of transistion from the current position

  7. After entering the values click on submit to observe the output.

Do follow the instructions (text is in red color) provided in the block at top-right corner of the page.

Any changes made using the features provided in display tab are reflected on the screen.

Below you have a Next button, click on it to open the world tab or else you have a Previous button to go back to homepage.
