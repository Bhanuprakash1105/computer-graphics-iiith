Controls

> Use only the arrow keys, number keys, mouse and F5 function key in the experiment.

> Refersh the page (F5), if you observe any random orientation of content.

> Hold the left mouse button and move the mouse for viewing in different angles.

> Hold the right mouse button and move the mouse for sliding on the screen.

Features

Click on the Start to observe the content of World Tab in which we can observe the description of the experiment.

Below we can observe World and Display buttons through which we can shift between the world and display tabs.

In the world tab many features are available:
  • We can see description of the experiment in the world tab.

  • Below the description we can see a block which contains labels of arm Palm, Wrist, Forearm, Elbow, Arm and Shoulder with check boxes and features for each of them.

  • In this each label part of the arm has Draw checkbox, Active checkbox and Transition feature.

  • Use Draw checkbox for adding a new respective part of the arm into the screen.

  • Use Active checkbox for adding the vertices of the respective part.

  • Use Transition feature for moving the respective part in distance of x-units in x-axis direction, y-units in y-axis direction and z-units in z-axis direction.

  • Inputs will not be accepted if it is not a number.

  • After entering the distance values in x, y and z transition boxes we need to click on SUBMIT button.

  • The Draw box must be checked for using Active box and Transiton feature.
In the diplay tab many features are available:
  • Set the check boxes value of Draw grid XY, YZ and ZX for adding or removing the grids respectively.

  • The Draw grid XY check box has default checked value but user can choose to check it or not.

  • The distance between two intersection points on the line is same in all the planes.

  • Use Show Animation checkbox for removing the animation or applying the animation.

  • Enter the coordinates in x, y and z boxes for chaniging the arm location.

  • Enter the position not the distance in (x, y, z).

  • Inputs will not be accepted if it is not a number.

  • Click on SUBMIT to observe the output on the screen for the shifting of arm location to the entered coordinates.

Any changes made using the features provided in display tab are reflected on the screen.
