Hierarchical Transformations
The Set-up: Articulated Arm

This experiment is designed to teach a hierarchical transformation model.
The world represented orderly as shown below. Each node is has 3 sections: Draw, Active and Transition.

Transition is the set of transformations applied to the position of structure w.r.t the current position / location of the structure.

All the features shown in display and world can be edited by clicking on them. Any changes to the tree are reflected on the display and vice-versa.

We constructed a hierarchical articulated arm model where each node/part follows a series of transformations from the root node to its current position. The top-most object in this hierarchy is the Shoulder followed by Elbow, Forearm, Wrist and Palm.

Some figures are coloured Red for differentiating them. The Draw box must be checked for using Active box and Transiton feature. After entering tansition values click on Submit
